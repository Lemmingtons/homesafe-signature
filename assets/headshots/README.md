# Team Headshots

Store team signature headshots here for the GitHub Pages app.

Recommended file format:

- Square JPG
- 360x360px or 720x720px
- Lowercase hyphenated filename, e.g. `ryan-lehmann.jpg`

Public URL pattern:

`https://lemmingtons.github.io/homesafe-signature/assets/headshots/ryan-lehmann.jpg`
