# Team Signature Workflow

Use this workflow when a VA creates signatures for Homesafe staff.

## 1. Prepare Headshots

- Crop each headshot square.
- Export as JPG at 360x360px or 720x720px.
- Use lowercase hyphenated filenames, e.g. `ryan-lehmann.jpg`.

## 2. Upload Headshots

Upload files into:

`assets/headshots/`

Once pushed to GitHub Pages, each image is public at:

`https://lemmingtons.github.io/homesafe-signature/assets/headshots/{filename}.jpg`

Example:

`https://lemmingtons.github.io/homesafe-signature/assets/headshots/ryan-lehmann.jpg`

## 3. Create Signature

Open:

`https://lemmingtons.github.io/homesafe-signature/`

Fill in the staff member's details.

Paste the hosted headshot URL into **Hosted Headshot URL**.

Use **Upload Photo** only as a fallback. Uploaded photos are embedded into the signature and must be compressed to stay inside Gmail's character limit.

## 4. Copy to Gmail

Click **Copy for Gmail** and paste into the user's Gmail signature settings.

Hosted headshots stay crisp because the signature references a public image URL instead of embedding a compressed base64 image.
